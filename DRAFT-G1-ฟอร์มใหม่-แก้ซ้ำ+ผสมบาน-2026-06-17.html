เสนอ "โครงฟอร์ม" ใหม่ · ของหลักโชว์เลย · ของไม่บ่อยพับ · ตัดของซ้ำ — ราคา/สูตร engine ไม่แตะ

① ปัญหาที่เจอตอนนี้ (สแกนของจริง)

❌ ออปชั่นซ้ำ 4 ตัว
ช่องหลักมี มุ้ง · มือจับ · กระจก · สีอลู อยู่แล้ว
แต่ "OPTION ทางเลือกลูกค้า" มีปุ่มซ้ำอีก:
เปลี่ยนกระจก เปลี่ยนผ้ามุ้ง เปลี่ยนสีอลู เปลี่ยนมือจับ
→ คนกรอกงง "ต้องกรอกตรงไหน" · ชื่อซ้ำ 2 ที่
❌ ของไม่บ่อยปนกัน

→ ใช้นานๆครั้ง แต่กองรวมใน "อุปกรณ์เสริม" — ควรจัดให้ของบ่อยเด่น ของไม่บ่อยพับลึก

＋ เสนอ OPTION ให้ลูกค้า (ทางเลือก +ราคา · ขึ้นในใบ) ▾ พับไว้
เลือก "เปลี่ยนอะไร" จากสเปคด้านบน (ไม่ต้องตั้งชื่อใหม่) + ใส่ราคาเพิ่ม → ขึ้นใบเป็น "OPTION : ..."
เปลี่ยนกระจกเปลี่ยนมุ้งเปลี่ยนสีอลูเปลี่ยนมือจับพิมพ์เอง
แก้ซ้ำ ของเดิม "เปลี่ยน X" ปนอยู่กับสเปคหลัก → ย้ายมารวมในกล่องนี้กล่องเดียว (สเปคหลักด้านบนเหลือที่เดียว ไม่ซ้ำ)
＋ อุปกรณ์เสริม (ใช้ไม่บ่อย) ▾ พับไว้
  • ครอบวงกบอลู (ไม่มี / 3 ด้าน / 4 ด้าน)
  • ฝังรางยู (กรีดพื้น) · ดรอปพื้น (ม.)
  • ธรณี (กันน้ำ / หลังเต่า+Drop Seal / หลังเต่า+สักหลาด / ไม่มี)
  • คาดตาราง (เส้นนอน/ตั้ง/โค้ง) · เสริมคานซัพพอร์ท
  • รื้อของเดิม
จัดใหม่ ของนานๆใช้ มารวมในกล่องพับนี้ — ฟอร์มหลักโล่ง กรอกเร็ว
＋ หมายเหตุ (ติ๊กมาตรฐาน + พิมพ์เอง) ▾ พับไว้
ติ๊กหลายข้อ (รวมในราคา/ไม่รวม/เงื่อนไข) — ตามที่ทำไปแล้ว ✅

③ สรุปสิ่งที่เปลี่ยน

  • แก้ซ้ำ 4 ตัว: มุ้ง/มือจับ/กระจก/สีอลู เหลือ ที่เดียว (สเปคหลัก) · "เปลี่ยน X" ย้ายไปรวมในกล่อง "เสนอ OPTION ให้ลูกค้า" กล่องเดียว
  • ของไม่บ่อยพับ: ครอบวงกบ/ฝังรางยู/ดรอปพื้น/ธรณี/คาดตาราง/เสริมคาน/รื้อ → กล่องพับ "อุปกรณ์เสริม"
  • ผสมบาน: ปุ่มเพิ่มบานย่อยเป็นชิป (ช่องแสงบน/บานข้างติดตาย/...) · แต่ละชิ้นกรอกขนาดเอง (ขนาดออกใบถูกแล้ว)
  • ลำดับ: ชนิด→รุ่น→ขนาด→สี/กระจก→มุ้ง→มือจับ → ผสมบาน → (พับ) OPTION ลูกค้า / อุปกรณ์เสริม / หมายเหตุ
  • ราคา/สูตร engine ไม่แตะ — งานจัดวาง/รวมปุ่มล้วน
⚠ จุดที่ต้องเคาะ: การ "เปลี่ยน X" (OPTION ลูกค้า) ตอนนี้ผูกกับการคิดราคาอัตโนมัติบางตัว — ถ้าย้ายรวมกล่องเดียว ต้องเช็คว่ายังคิดราคาถูก · เป็นงานที่ต้องคุยกับ dev (แชทกลาง)
