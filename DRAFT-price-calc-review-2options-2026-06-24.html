🧮 หน้าสรุป "วิธีคิดราคา" สำหรับเฟิร์นตรวจ — ดราฟ 2 แบบ

ทั้ง 7 กลุ่ม (G1–G7) · วิธีคิดราคา = ให้เฟิร์น (คนประเมินหลัก) ตรวจละเอียดก่อนส่งลูกค้า · ต่างจาก แยกราคาแต่ละส่วน (อันนั้นไว้ส่งลูกค้า)
❓ งานเดิมพังไหม?  ไม่พัง — เป็นแค่ "มุมมองอ่านอย่างเดียว" ที่ดึงค่าจาก engine ที่คิดอยู่แล้ว (เรต/ขนาด/min/ปัดเศษ/ออปชั่น) ไม่แตะสูตรราคา → golden-snapshot 150/150 คงเดิม · เป็นของเพิ่ม (ปุ่ม/โหมด) ไม่ทับของเก่า
❓ ยาก/ช้าไหม?  ปานกลาง — กุญแจคือต้องให้ calcUnit "คาย" ขั้นตอนคิด (เรต × พื้นที่ → min → ปัดเศษ → +ออปชั่นแต่ละตัว) ออกมาโชว์ · ตอนนี้ engine คาย breakdown บางส่วนแล้ว (จากฟีเจอร์แยกราคา/การ์ดราคา) → ต่อยอดได้ ไม่ได้เริ่มศูนย์
❓ ทำทีละกลุ่มหรือทีเดียว?  ทีเดียวครบ 7 กลุ่ม — เพราะ engine ราคาเป็นก้อนเดียวร่วมกัน (calcUnit/genQuote) → ทำที่จุดเดียว ใช้ได้ทุกกลุ่ม (ข้อดีของ shared engine)

อยู่ในหน้าทำใบเลย · เฟิร์นกด 🔍 ตรวจวิธีคิด → ตรวจเสร็จสลับกลับ 👤 ลูกค้า แล้วพิมพ์ได้เลย
👤 ลูกค้า (ปกติ) 📊 แยกราคาแต่ละส่วน 🔍 วิธีคิดราคา (เฟิร์นตรวจ)
ใบเสนอราคา — งานบาน + ระแนง (ตัวอย่าง 2 รายการ)
1) บานเลื่อน เซมิยูโร (1.8×2.1) × 3 บาน76,100
• อลู สีเทาซาฮาร่า · กระจกเขียว 6 มม. · ล็อก+กุญแจ · มุ้งจีบ · คาดตาราง
• ตัวบาน = 62,400 · ล็อก+กุญแจ = 3,500 · มุ้งจีบ = 9,000 · คาดตาราง = 1,200
🔍 วิธีคิดราคา ภายใน — ไม่พิมพ์ให้ลูกค้า
รุ่น บานเลื่อน เซมิยูโร · เรต 5,500 บาท/ตร.ม.
พื้นที่/บาน = 1.8 × 2.1 = 3.78 ตร.ม. × 3 บาน = 11.34 ตร.ม.
ฐาน = 5,500 × 11.34 = 62,370
เช็ค min: max(min 0 , 62,370) = 62,370 → ปัดเศษขึ้นร้อย = 62,400
+ ล็อก+กุญแจ = 3,500 · + มุ้งจีบ (3 บาน) = 9,000 · + คาดตาราง = 1,200
= ยอดสุทธิ 76,100 บาท
2) ระแนงบังตา 1"x1.6" หน้า 1" รวมโครง (3.0×2.5)36,800
• อลู สีดำ · ระยะห่าง 5 ซม. · รวมโครง
• ระแนง+โครง = 36,800
🔍 วิธีคิดราคา ภายใน — ไม่พิมพ์ให้ลูกค้า
รุ่น ระแนงบังตา รวมโครง · เรต 4,900 บาท/ตร.ม.
พื้นที่ = 3.0 × 2.5 = 7.5 ตร.ม.
ฐาน = 4,900 × 7.5 = 36,750 → ปัดเศษขึ้นร้อย = 36,800
= ยอดสุทธิ 36,800 บาท
รวม 2 รายการ = 112,900 บาท (ก่อน VAT)
↑ กดปุ่มโหมดด้านบนดูได้จริง · ตอนนี้ = 👤 ลูกค้า (ปกติ)
ข้อดี: อยู่หน้าเดียวกับใบ · เฟิร์นตรวจเสร็จสลับพิมพ์ให้ลูกค้าได้เลย · เร็ว · ทำครั้งเดียวใช้ทุกกลุ่ม
ข้อควรระวัง: ข้อมูลแน่นในหน้าเดียว (กันพลาดด้วย warn "ไม่พิมพ์ให้ลูกค้า" + โหมดวิธีคิดไม่ติดไปเวลาพิมพ์)
