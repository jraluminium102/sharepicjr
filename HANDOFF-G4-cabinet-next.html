PART A — แก้ราคา: Aztec gray = 950 (stock)

มติพี่นัท (ใหม่): Aztec gray (สีหลัก idx 4) เมื่อเป็นของ stock ค่าสีอลูทึบ = 950/ตร.ม. (ไม่ใช่ 1,600 ที่ตั้งไว้)
950 อยู่ระหว่างซาฮารา 400 กับอบพิเศษ 1,600 · ตรงกับตาราง o-bsfinish (ระแนงเลื่อน ลูกฟูก 2 ทาง) ที่ Aztec=950 อยู่แล้ว

แก้ rn90FinRate (บ.~933) — แยก idx4 ออกจากกลุ่มอบพิเศษ:

// เดิม: if(ci===4||ci===10||ci===12) return 1600;
function rn90FinRate(ci){
  if(ci===2||ci===3) return 400;
  if(ci===4) return 950;                    // ← เพิ่ม: Aztec gray (stock) แยกออกมา
  if(ci>=5&&ci<=7) return 1500;
  if(ci===8||ci===9||ci===11) return 2200;
  if(ci===10||ci===12) return 1600;         // อบพิเศษ / สีชุบ (เอา 4 ออก)
  return 0;
}

#จุดปรับทำอะไรระดับ
1จับ "วัสดุผนัง + ชนิดชั้น" เป็นบล็อกเดียวการ์ดวัสดุผนัง (อลูทึบ/กระจก) อยู่บน → ชิปชนิดชั้นอยู่ใต้ทันที + ข้อความ cascade "ชั้นกระจก = ไม่คิดค่าสี" โผล่เมื่อเลือกชั้นกระจกคุ้มสุด
2สี → โชว์ผลกระทบ cascadeใต้ dropdown สี แสดง hint ตามวัสดุผนังที่เลือก: ผนังอลู="ค่าสี rn90" / ผนังกระจก="colorPrice เต็ม" · ย้ำว่าสีนี้กระทบ "เฉพาะโครง" ไม่ใช่บานหน้าสูง
3ลึก + จำนวนชั้น จัดแถวเดียววาง 2 ช่องในแถวเดียว badge auto ชัดทั้งคู่ — สื่อว่าข้ามได้สูง
4ชิป switch "ตู้อลู / ฝาตู้" บนสุดเพิ่มชิป 2 ตัวบนสุดของฟอร์ม (sync กับ .i-prod) → รู้ทันทีกำลังกรอกสินค้าอะไร section ล่างเปลี่ยนตามคุ้มสุด
5hint ใต้ "จำนวนบาน"ตู้อลู: "ยิ่งหลายบาน บานยิ่งเล็ก ราคา/บานลด" · ฝาตู้: "บานที่ 2-3 +5,500/บาน"เสริม
6ผนังหลัง checkbox tap zone ใหญ่แยก label "+ผนังหลัง" + hint "ปกติชนผนังปูน" 2 บรรทัด · row คลิกได้ทั้งแถบ ≥44pxเสริม

pattern: ชิป switch สินค้า (ข้อ 4) ใช้ chSetChip sync กับ .i-prod เดิม + trigger buildItemOpts · อย่า hardcode รายการสินค้า อ่านจาก dropdown เดิม

⚠ PART C — เตือน: ห้ามลบของเดิม (ตรวจแล้ว)

ดราฟ UX โชว์เฉพาะ ช่องหลัก เพื่อความชัด — ไม่ได้แปลว่าให้ตัดของอื่นทิ้ง. generic เหล่านี้ยังต้องอยู่ในฟอร์มตู้ตามเดิม:
  • มีคาดตาราง (o-gridmark) — พี่นัทเก็บไว้ (ไม่ตัด)
  • งานเสริม/วิศวกรรม (o-extrawork) · รื้อของเดิม (o-removeold)
  • หมายเหตุ/OPTION (i-note) + ปุ่มหมายเหตุมาตรฐาน (o-remark-add)
  • สีอลู (i-color) — header
ตัดของไม่เกี่ยวออกครบแล้ว (ยืนยันในโค้ดปัจจุบัน — ไม่ต้องทำซ้ำ):
  • ฝังรางยู o-uchannel → ตัดแล้ว (&& !p.cabinet && !p.future_tech)
  • ดรอปพื้น o-dfm + ครอบวงกบ o-fcsides → ไม่ render (ตู้เป็น areaOnly)
  • ประตู/หน้าต่าง i-type → ซ่อนแล้ว (ITYPE_NA มี 'ตู้อลู'+'ฝาตู้')
