ดราฟ ② ลากจัดลำดับฟีเจอร์ — แยกต่อกลุ่ม ตรง render จริง

หมวด + ลำดับ + ของข้างใน = ดึงจากเว็บจริงปัจจุบัน (ไม่ได้เขียนมือ) · เลือกกลุ่ม → ลาก/▲▼ จัดลำดับ → กด "กางดู" เห็นออปชั่นข้างใน

ทำงานยังไง: เพิ่ม map GROUP_ORDER[กลุ่ม]=[ลำดับหมวด] · groupGHOpts เรียงตามของกลุ่มที่เปิด
กลุ่มอื่นไม่พัง: จัด G2 = แก้แค่ GROUP_ORDER['2'] · ล็อกด้วย order-snapshot (เด้งแดงถ้าเพี้ยน)

ขอเคาะ: 1) ตรงของจริงแล้วใช่ไหม · 2) จัด G2 ระแนง ลำดับที่อยากได้ → บอกผม ส่ง dev

⚠️ จุดที่ต้องรู้ (ความจริงจาก render):
• G1 ไม่มีหมวด "ล็อค·มุ้ง·คาดตาราง" ในลิสต์นี้ — เพราะ G1 ดัน ล็อค/คาดตาราง ไปไว้ นอก accordion (กล่องเด่นด้านบน) คนละระบบกับการจัดหมวดนี้ · ถ้าอยากจัดอันนั้นด้วย = งานแยก
• G5 มุ้ง = ฟอร์มเฉพาะ ไม่มีหมวด · G3 = หลังคา + ฝ้า-ผนัง รวมกัน (9 หมวด)
• "กางดู" = ออปชั่นจริงในหมวด · ถ้าเห็นออปชั่น อยู่ผิดหมวด บอกได้ (ย้ายข้ามหมวด = งานแยก)
