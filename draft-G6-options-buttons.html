จุดต่างจากเดิม (After)

  • ปุ่มชุด "+ เพิ่มบาน" / "+ เพิ่มหลังคา" — แบนชั้นเดียว ≥44px ชัด
  • ชนิดการเปิด + ราง ขึ้นฟอร์มหลัก (กล่องฟ้า) — ไม่ต้องกดเปิด opts
  • ชุดล็อค ขึ้นฟอร์มหลัก (ชิป 2) — ไม่ซ่อนใน opts
  • มือจับ → กล่องรวบ toggle แบรนด์ + conditional sub
  • มุ้ง → เลือกชนิดก่อน → โชว์ ผ้า/สีกรอบ/บาน
  • ครอบวงกบ / แผ่นทึบล่าง / รางยู → รวมหมวด "อื่นๆ" เดียว
  • สีลูกฟูก โผล่เฉพาะเลือก "อลูลูกฟูก" (conditional)
  • opts summary text ชัดกว่าเดิม — บอกว่ามีอะไรข้างใน

ปุ่มเดิม ปุ่ม "เพิ่มส่วน" ใช้ร่วมกันทั้งระบบ บางทีไม่รู้ว่า GH ต้องกดตรงไหน

ด้าน A opts ซ้อน details

opts เดิมจัดเป็น 4 <details> พับ — ทุก control รวมอยู่ข้างใน รวมทั้ง ชนิดการเปิด/ราง/ชุดล็อค ที่ใช้ทุกงาน

🚪 ชนิด / การเปิด (ซ่อนอยู่)
🔧 มือจับ / ล็อค
มือจับ Cmech / ดิจิตอล / สแตนอร่าม ต่างมี dropdown + ช่องกรอกแยก → แสดงพร้อมกันทุก dropdown แม้ไม่ใช้
🪟 มุ้ง
เลือกมุ้งจาก dropdown 13 รุ่น → โผล่ช่องกว้าง/สูง/บาน/ผ้า/สีกรอบพร้อมกัน ยาว
➕ อื่นๆ (ครอบวงกบ/ดรอปพื้น/หมายเหตุ)
ครอบวงกบ / ฝังรางยู / แผ่นทึบ / สีลูกฟูก / งานเสริม / รื้อของเดิม / ดรอปพื้น อยู่รวมกัน ไม่ conditional

ปัญหาเดิม: ชนิดการเปิด+ราง+ชุดล็อค ซ่อนอยู่ใน opts ต้องเปิด details ก่อน · มือจับ dropdown ทุกตัวโผล่พร้อมกัน · มุ้งไม่ conditional
