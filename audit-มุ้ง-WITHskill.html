1. สรุปผล (อ่านอันเดียวจบ)

ระบบ ครอบคลุมชนิดมุ้ง “ราคาแยกชุด” ได้ดีมาก (มุ้งจีบ, มุ้งจีบนิรภัยสแตน, มุ้งม้วน/เตะได้, มุ้งแม่เหล็ก, มุ้งกันแมว/หนู, blackout/ทูเวย์, เฟรมเล็ก/ใหญ่/ติดตาย) ทั้งแบบสินค้าเดี่ยว (หมวดมุ้ง) และแบบ add-on ในบาน (o-mosq)

แต่มี ช่องว่างสำคัญ 3 จุด (🔴) ที่ทำให้ออกใบไม่ตรงใบจริง:
① “มุ้งนิรภัย” (เฟรมตัน ผ้าสแตน) ไม่มีเป็นสินค้า/ออปชั่น ทั้งที่เป็นมุ้งที่ใบจริงใช้บ่อยที่สุด — ระบบมีแต่ “มุ้งจีบนิรภัย” (คนละตัว)
② “เส้นคาดมุ้ง” ไม่มีช่องเลือก + ไม่ออกลงใบ ทั้งที่ QC ระบุเป็นสเปกบังคับของเฟรมเล็ก/ใหญ่/นิรภัย
③ add-on มุ้งในบาน ออกข้อความลงใบไม่ครบ (ไม่โชว์สีกรอบมุ้ง/เส้นคาด/แบบยัด-เสริมวงกบ ในบรรทัดรายละเอียด)

2. คลังคำศัพท์ “มุ้ง” ที่เจอในใบจริง (เกณฑ์เทียบ)

ชนิดมุ้ง (ใบจริงเขียน)รายละเอียด/ออปชั่นที่ใบจริงระบุพบในใบ
มุ้งนิรภัย (เฟรมตัน)เฟรมลายไม้สีไวท์โอ๊ค / เฟรมสีเทาซาฮาร่า · บางใบเขียน “ประตูมุ้งนิรภัย”บุญญารัตน์ (ข้อ 2,5-7,17-18), ชวลิต, ปีเตอร์
มุ้งเฟรมเล็กผ้าไฟเบอร์เทา/ดำ · เฟรมสีเทาซาฮาร่า · มีเส้นคาดมุ้งบุญญารัตน์, สุจินต์
มุ้งเฟรมใหญ่ผ้าไฟเบอร์ดำปีเตอร์ (ทุกข้อ)
มุ้งจีบเฟรมลายไม้ไวท์แบมบู / เฟรมสีเทาซาฮาร่า · ผ้าไฟเบอร์ดำชวลิต, สุจินต์
มุ้งจีบนิรภัยเปิดได้ 1 ทาง · รางล่าง 2.5 ซม.QC อิงควัชร์
มุ้งม้วนกำหนดขนาดตามรูป · ขั้นต่ำ 2 ตร.ม./ชุด แยกกลางQC อิงควัชร์
เส้นคาดมุ้ง (ออปชั่นร่วม)เฟรมเล็กเปิด=1เส้น · เลื่อนทั่วไป=คู่ · SMS เสาโหนก=0/1 · เฟรมใหญ่เลื่อน=ไม่ใส่ · นิรภัยสูง>2ม.=คาด 0.90ม.QC อิงควัชร์ (ละเอียดสุด)
ออปชั่นติดตั้งยัดในวงกบ / นอกวงกบ · เสริมวงกบ · กว้าง>1.5ม. เสริมเส้นซอยQC อิงควัชร์

3. ตารางเทียบ (ใบจริง ↔ ระบบ)

รายการ (ใบจริง) ใบจริงเขียนอะไร ระบบมี option + ออกครบไหม ขาด/ผิด/ต้องแก้ + root cause สถานะ
มุ้งนิรภัย (เฟรมตัน ผ้าสแตน) “+ มุ้งนิรภัย (เฟรมลายไม้สีไวท์โอ๊ค)” — มุ้งยอดฮิตในใบจริง ❌ ไม่มีสินค้า/ออปชั่นชื่อนี้เลย · ระบบมีแต่ “มุ้งจีบนิรภัย” (สแตน 0.8 จีบได้ คนละแบบ) · มีตารางราคา IMP24-27 (2,500/2,900/3,000/3,500) แต่ ไม่มี product ผูกใช้ (orphan) 🔴 จัดทำสินค้า/addon “มุ้งนิรภัย” ใหม่ ผูก IMP24-27 · root: product ขาด (ตารางราคามีแล้วแต่ลืมต่อ product) 🔴 ขาด
เส้นคาดมุ้ง QC บังคับระบุ: เฟรมเล็กเปิด 1 เส้น / เลื่อนคู่ / นิรภัยสูง>2ม. คาด 0.90ม. ฯลฯ ❌ ไม่มีช่องเลือกเส้นคาดมุ้งใน o-mosq / สินค้าหมวดมุ้ง · genQuote ไม่ออกข้อความเส้นคาด (det = name+ผ้า+สีกรอบ เท่านั้น) · มีแค่ checklist เตือนข้อความ (line 1575) 🔴 เพิ่ม dropdown เส้นคาด (ไม่ใส่/1เส้น/คู่/0.90ม.) + ออกลงใบ · root: option ขาด + genQuote ไม่ดึงลงใบ 🔴 ขาด
add-on มุ้งในบาน — ข้อความลงใบ ใบจริงเขียนใต้บาน เช่น “- มุ้งจีบ (เฟรมสีเทาซาฮาร่า/ผ้าไฟเบอร์ดำ)” ⚠ ระบบบวกราคา + ออกบรรทัดได้ แต่ det = name + ผ้า + สีกรอบ เท่านั้น · ไม่โชว์สีกรอบเป็นวงเล็บแบบใบจริง และไม่มีเส้นคาด/แบบยัด-เสริมวงกบ 🔴 ปรับ genQuote ให้ออกฟอร์แมต “(เฟรม.../ผ้า...)” ตรงใบจริง · root: genQuote ออกข้อความไม่ครบ 🔴 ผิดรูป
ผ้ามุ้ง default (เทา vs ดำ) ใบจริงส่วนใหญ่ใช้ ผ้าไฟเบอร์ดำ (ปีเตอร์/ชวลิต/สุจินต์ ทุกข้อ) ⚠ ระบบ default = fiber_gray (ไฟเบอร์เทา) — สวนทางการใช้จริง 🟡 พิจารณาตั้ง default เป็นไฟเบอร์ดำ (เช็คซ้ำกับพี่นัท ว่าค่าตั้งต้นควรเป็นอะไร) 🟡 ตั้งต้น
สีกรอบมุ้ง (ลายไม้) ใบจริงมี “เฟรมลายไม้สีไวท์โอ๊ค”, “ไวท์แบมบู” (สีลายไม้เฉพาะ) ⚠ o-mosqcolor มี: ขาว/ดำ, ซาฮาร่า, เคแอลอบพิเศษ, เคแอลลายไม้ · มี “ลายไม้” แต่ ไม่มีเฉดไวท์โอ๊ค/ไวท์แบมบู ระบุชื่อ 🟡 ตรวจว่า “เคแอลลายไม้” ครอบคลุมเฉดที่ใบจริงใช้ไหม (เช็คซ้ำ) 🟡 เช็คซ้ำ
แบบติดตั้ง ยัด/เสริมวงกบ QC ระบุ “ยัดในวงกบ/นอกวงกบ”, “เสริมวงกบ” ⚠ มีเฉพาะ “เสริมวงกบ” ใน 2 สินค้า (imp32, mj_kick ติดล้นนอกวงกบ) · add-on มุ้งในบาน ไม่มีช่องเลือกยัด/เสริม 🟡 เพิ่มช่องเลือกแบบติดตั้งใน addon (เช็คซ้ำว่าจำเป็นออกใบไหม) 🟡 ไม่ครบ
ประตูมุ้งนิรภัย (บานรอง) บุญญารัตน์: “บานรอง: ประตูมุ้งนิรภัย” ⚠ เป็นกรณีของ มุ้งนิรภัย ที่ทำเป็นบานประตู — ผูกกับ 🔴 ข้อ 1 (ไม่มีมุ้งนิรภัย) 🟡 ครอบคลุมเมื่อแก้ข้อ 1 (มุ้งนิรภัยเป็น product/addon บานประตูได้) 🟡 พ่วงข้อ1
เสริมเส้นซอยกลาง (กว้าง>1.5ม.) QC: บานกว้างเกิน 1.5 ม. ต้องเสริมเส้นซอยกลาง ⚠ ไม่มีออปชั่น/ออโต้คิดเส้นซอย · มีแต่ checklist เตือน 🟡 เพิ่มเตือน/คิดราคาเส้นซอยอัตโนมัติ (เช็คซ้ำว่าคิดเงินเพิ่มไหม) 🟡 ไม่ครบ
มุ้งจีบผ้าไฟเบอร์ดำ · เฟรมซาฮาร่า/ลายไม้✅ มีครบ: addon (mj_sd_basic) + สินค้าเดี่ยว · render ออกใบได้ (FULL-B row 4-19,34-42)—🟢 ตรง
มุ้งเฟรมเล็ก / ติดตายผ้าไฟเบอร์ · เฟรมซาฮาร่า✅ imp21/imp22 มีทั้ง addon + สินค้าเดี่ยว (FULL-B row 75-76)—🟢 ตรง
มุ้งเฟรมใหญ่ผ้าไฟเบอร์ดำ✅ imp23 มี (FULL-B row 77)—🟢 ตรง
มุ้งจีบนิรภัย (สแตน 0.8)เปิด 1 ทาง · รางเตี้ย✅ imp31/imp32 + mj_screen_safety (FULL-B row 81-82,93)—🟢 ตรง
มุ้งม้วน / เตะได้ตามขนาด · ขั้นต่ำ/ชุด✅ imp29 + mj_kick_150/300/600 (FULL-B row 79,89-91)—🟢 ตรง
มุ้งแม่เหล็ก / กันแมว / กันหนู / blackout / ทูเวย์ชนิดเสริม✅ imp30/33/35, mj_blackout, mj_sd_twoway, mj_keep_* (FULL-B row 80,83-88,92)—🟢 ตรง

4. รายการ “ต้องเพิ่ม/แก้” จัดตามความสำคัญ

  • P1 เพิ่ม “มุ้งนิรภัย” (เฟรมตัน ผ้าสแตน) เป็นสินค้า/addon — ผูกตารางราคา IMP24-27 ที่มีอยู่แล้วแต่ยังลอย (orphan). เป็นมุ้งที่ใบจริงใช้บ่อยสุด ขาดไป = ออกใบไม่ได้
  • P1 เพิ่ม “เส้นคาดมุ้ง” เป็น dropdown (ไม่ใส่/1เส้น/คู่/คาด 0.90ม.) ทั้งใน addon บาน และสินค้าหมวดมุ้ง + ให้ genQuote ออกข้อความลงใบ (ตอนนี้มีแค่ checklist เตือน)
  • P1 แก้ genQuote บรรทัด addon มุ้ง ให้ออกฟอร์แมตตรงใบจริง: “มุ้งจีบ (เฟรมสีเทาซาฮาร่า/ผ้าไฟเบอร์ดำ)” — ปัจจุบันต่อ string เฉยๆ ไม่จัดเป็นวงเล็บ
  • P2 ทบทวน default ผ้ามุ้ง (ปัจจุบัน=ไฟเบอร์เทา · ใบจริงใช้ดำเป็นหลัก)
  • P2 เพิ่มช่อง แบบติดตั้ง ยัด/เสริมวงกบ ใน addon มุ้งในบาน
  • P3 ตรวจเฉดสีกรอบลายไม้ (ไวท์โอ๊ค/ไวท์แบมบู) ว่าครอบใน “เคแอลลายไม้” พอไหม
  • P3 ออโต้คิด/เตือน “เส้นซอยกลาง” เมื่อบานกว้าง > 1.5 ม.
