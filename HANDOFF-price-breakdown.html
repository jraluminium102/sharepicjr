3) กฎเหล็ก (ห้ามพลาด)

ผลรวมต้องเท่าเดิมΣ priceLines = it.r.sell เป๊ะ (แค่ "แสดงการแตก" ไม่เปลี่ยนยอด) · ราคารวม/VAT/ยอดสุทธิ ไม่เพี้ยน
รวมราคา (ไม่ split)โหมด "รวมราคา" คงเดิม โชว์ก้อนเดียว · แตกเฉพาะตอนกด "แยกราคา"
component = 0ไม่ต้องโชว์บรรทัด (เช่น สีอบขาว = สีฟรี colorUp=0 → ไม่ขึ้น "เพิ่มสีอลู")
ครบทุกชนิดสินค้าทำให้ generic — ทุกบาน/หลังคา/ตู้/ฯลฯ ที่มี component ราคา ต้องแตกได้ ไม่ใช่เฉพาะบานเลื่อน
📋 Definition of Done:
  1. กด "แยกราคา" → เห็นทุก component ราคา: ฐานบาน · เพิ่มสีอลู · เพิ่มกระจก · คาดตาราง · มือจับ · เสริมคาน · แผ่นทึบ · มุ้ง · ฯลฯ ครบ
  2. Σ component = ยอดบานย่อยเดิม (ทุกรายการ) · node test/quote-fidelity.mjs PASS (ยอด/VAT ไม่เพี้ยน)
  3. สีอลู/กระจก upcharge ที่เคยหาย → โชว์แล้ว (เคสสีพิเศษ/กระจกอัปเกรด)
  4. component=0 ไม่โชว์ · โหมดรวมราคาเดิมไม่กระทบ
  5. พี่นัทกดแยกราคาแล้วเห็นครบทุกรายการ
