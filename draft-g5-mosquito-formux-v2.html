หมวด accordion ที่เสนอสำหรับมุ้ง (p.cat==='มุ้ง')
1. ฟอร์มหลัก (เห็นทันที): ประเภท · ขนาด · จำนวนบาน
2. การ์ดส้ม (โหมด B): ติดมุ้งบานเดิม + ค่าเสริม
3. ชิปหลัก: ผ้ามุ้ง + สีผ้า (เฉพาะ screen_addon)
4. ชิปลักษณะการเปิด (label-only · imp21/23)
5. Accordion "อุปกรณ์เสริม": สีกรอบมุ้ง + ครอบวงกบ + งานเสริม
6. Accordion "หมายเหตุ": dropdown preset + textarea
จุดที่ยังต้องแก้ใน index.html (ดู HANDOFF v3):
- โหมด B ยังอยู่กลางฟอร์ม → ย้ายขึ้นบน
- ผ้ามุ้ง/สีผ้า ยังเป็น dropdown → แปลงชิป
- สีกรอบมุ้ง (o-mscolor) อาจหลุดเข้า bucket 🚪 เพราะ keyword 'สี ' → แก้ CATS
