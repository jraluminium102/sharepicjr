เงื่อนไขการเข้าทำงาน

  1. ไฟฟ้าและน้ำประปา ลูกค้าเป็นผู้ดำเนินการจัดหา
  2. สำหรับหน้างานที่มีการรื้ออลูมิเนียม มีรายละเอียด ดังนี้
    1. 2.1 ไม่เก็บงานปูน รอยแตกร้าว รอยบิ่น
    2. 2.2 อาจมีเสียงดังระหว่างการผ่างานปูน
    3. 2.3 อาจมีฝุ่นเป็นจำนวนมากที่เกิดขึ้นจากงานผ่าปูน
  3. การขออนุญาตเข้าทำงานในพื้นที่และการวางเงินค้ำประกันก่อนเข้าทำงาน ลูกค้าเป็นผู้ดำเนินการจัดหา (กรณีเริ่มงานแล้วเกิดเหตุสุดวิสัยระหว่างทำงานทำให้ไม่สามารถทำงานต่อได้ บริษัทขอสงวนสิทธิ์คิดค่าดำเนินการตามจริง)
  4. นอกเหนือจากราคาที่เสนอจะคิดเป็นงานเพิ่มทั้งหมด และจะต้องเสนอราคางานเพิ่มก่อนเริ่มดำเนินการ
  5. เงื่อนไขการรับประกันผลงาน การติดตั้ง 1 ปี
    โดยการรับประกันผลงานการติดตั้งอุปกรณ์ดังกล่าวเช่น บานตก, บานไม่ลื่น, น้ำรั่วและอื่นๆ นับจากวันที่ส่งมอบงาน ผู้รับจ้างขอสงวนสิทธิ์ไม่รับประกัน ในกรณีที่เกิดความชำรุดเสียหายอันเนื่องมาจากการทรุดตัวของพื้นดิน, การใช้งานผิดประเภท, อุบัติเหตุ, ภัยธรรมชาติ รวมถึงไม่ครอบคลุมความเสียหายน้ำรั่วซึมที่เกิดจากผนังปูนเดิม, บัวเดิม หรือหลังคาเดิมที่มีการแตกร้าวหรือชำรุด ทั้งนี้ผู้ว่าจ้างตกลงจะไม่ติดใจเรียกร้องค่าเสียหายหรือค่าชดเชยใดๆ กับผู้รับจ้างในทุกกรณี
    ทั้งนี้การติดตั้งผนังเบากรุสมาร์ทบอร์ดเป็นวัสดุที่อาจเกิดรอยแตกร้าว (Cracks) ตามคุณสมบัติของวัสดุและสภาพแวดล้อม อย่างไรก็ตาม ผู้รับจ้างยังคงรับประกันคุณภาพงานติดตั้งในส่วนของปัญหาน้ำรั่วซึม อันเกิดจากการติดตั้งของผู้รับจ้าง ภายใต้เงื่อนไขการรับประกันที่ระบุไว้ข้างต้น

เงื่อนไขแบบและใบเสนอราคา

  1. 1) หากลูกค้าต้องการปรับเปลี่ยนสีวัสดุอุปกรณ์หรือการปรับแก้ไขแบบ ต้องแจ้งข้อความผ่านช่องทาง LINE Official, Facebook เท่านั้น และสามารถแก้ไขได้ก่อนภายในวันที่มัดจำ
  2. 2) หากลูกค้าต้องการแบบ 3D จะมีค่าใช้จ่ายเพิ่มเติม (มี 2 แพ็กเกจ) ซึ่งลูกค้าสามารถส่ง Reference ที่ต้องการมาก่อนการออกแบบได้ แต่ถ้าไม่มี Reference บริษัทจะทำการออกแบบตามแบบ 2D ของบริษัท
    1. 2.1) กรณีที่ยอดรวมงานมากกว่า 200,000 บาท สามารถนำค่าแบบ 3D มาใช้เป็นส่วนลดได้เต็มจำนวน
    2. 2.2) กรณีที่ยอดรวมงานน้อยกว่า 200,000 บาท จะไม่สามารถนำค่าแบบ 3D มาใช้เป็นส่วนลดได้
    3. 2.3) แบบ 3D สามารถแก้ไขได้ไม่เกิน 3 ครั้ง
  3. 3) หากลูกค้าต้องการให้บริษัทจัดซื้อและติดตั้งอุปกรณ์เครื่องใช้ไฟฟ้าต่างๆ เช่น ปลั๊กไฟ, สวิตช์ไฟ, ชุดโคมไฟพร้อมหลอด, พัดลมระบายอากาศ หรืออื่นๆ รูปแบบจะเป็นไปตามบริษัทกำหนดและจัดหาได้
  4. 4) หากลูกค้าต้องการให้บริษัทจัดซื้อสีสำหรับใช้ทาผนังเบากรุสมาร์ทบอร์ดและสำหรับงานเก็บสี เฉดสีจะเป็นไปตามมาตรฐานของบริษัทที่จัดหาได้ ซึ่งจะใกล้เคียงกับสีเดิม (อาจจะไม่เหมือนสีเดิม)
  5. 5) ถ้าลูกค้าต้องการธรณีประตูเรียบเสมอพื้น (ดรอปพื้นฝังธรณี) สามารถใช้ระยะดรอปพื้นจากแบบ 2D ส่งให้ผู้รับเหมาดำเนินการดรอปพื้น ก่อนช่างของบริษัทเข้าวัดหน้างานจริงก่อนผลิต
  6. 6) หลังจากมัดจำและหน้างานดรอปพื้น ก่อฉาบช่องเปิดเรียบร้อยแล้ว ทีมช่างผลิตของบริษัทจะเข้าวัดหน้างานจริง ซึ่งจะไม่สามารถปรับเปลี่ยนแบบหรือแก้ไขขนาดพื้นที่ติดตั้งหน้างานทุกกรณี เช่น ปูกระเบื้อง, ทำฝ้า, ฉาบปูน และอื่นๆ
