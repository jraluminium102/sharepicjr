📄 ใบเสนอราคาออกใบจริงส่งลูกค้า (มีหัวบิล/เงื่อนไข/VAT ครบ)

📄 ออกใบเสนอราคา — 5 ขั้น

1
แตะแท็บ “ใบเสนอราคา” ด้านบน → กด “＋ เพิ่มส่วน”
2
เลือกกลุ่มงาน (ปุ่มแถวบนสุด): บาน / ระแนง·ราว / หลังคา / ตู้ / มุ้ง / กั้นห้อง / ม่านซิป
3
เลือกรูปแบบ (ปุ่มแดง) → ถ้าเป็นระแนง/หลังคา จะมีปุ่มย่อยโผล่ทีละขั้น ให้กดไล่ลงไป (ขนาดเส้น → ระยะโชว์ → โครง)
4
กรอก กว้าง × สูง (เมตร) + จำนวนบาน (กดปุ่ม − / +) + จำนวนชุด · เลือกสี/กระจกในแถบฟ้า
5
กรอกลูกค้า/วันที่ (แถบ “ข้อมูลลูกค้า”) → กดปุ่มแดง “ส่งเข้าระบบ JR” = ได้ใบจริง
💡 เพิ่มหลายรายการในใบเดียว: กด “＋ เพิ่มส่วน” ซ้ำ · ยอดรวม+VAT คิดให้อัตโนมัติ

🦟 กั้นห้องกระจก (หลายด้าน)

  • เลือกกลุ่ม “กั้นห้อง” → ระบบทำ “ด้าน A” ให้
  • กด “＋ เพิ่มด้าน” เพื่อเพิ่มด้าน B, C… · กด “＋ เพิ่มชิ้นในด้านนี้” ถ้า 1 ด้านมีหลายชิ้น (ระแนง+ประตู+ช่องแสง)
  • สี/กระจก ตั้ง “ทั้งใบ” ทีเดียว ทุกด้านตามหมด

⚡ คิดเร็ว — 3 ขั้น

1
แท็บ “คิดเร็ว” → กด “＋ เพิ่มชุด”
2
กดปุ่มกลุ่มงาน → กดปุ่มรูปแบบ (ถ้าไม่มีในปุ่ม กด “อื่นๆ ▾”)
3
กรอก กว้าง×สูง + เลือกสี/กระจก → ได้ราคาทันที มุมขวา
⚡ คิดเร็ว = ดูราคาในเครื่องเท่านั้น · จะออกใบจริงต้องไปแท็บ “ใบเสนอราคา”

✎ แก้ข้อความ/ราคา ในใบ

หลังออกใบ กดปุ่ม ✎ แก้ทุกข้อความ แล้ว คลิกที่คำไหนก็พิมพ์ทับได้เลย (ขอบจุดประแดง = แก้ได้):
  • หัวเรื่อง · ชื่อรายการ · เงื่อนไขท้ายใบทั้งหมด · ป้ายต่างๆ
  • ราคาต่อแถว — พิมพ์เลขใหม่ → ยอดรวม/VAT คิดใหม่ทันที
จำให้: ข้อความที่แก้ (เช่นเงื่อนไข) จำไว้ในเครื่อง ใบหน้าถัดไปใช้ต่อเลย · อยากกลับค่าเดิมกด ↺ คืนข้อความเดิม
💡 แก้เสร็จกด ✓ เสร็จการแก้ · เซฟอัตโนมัติ ไม่ต้องแก้โค้ด

🎨 ออปชั่นทางเลือกลูกค้า

ในแต่ละรายการ กด “＋ OPTION ทางเลือกลูกค้า” → เลือกเปลี่ยนกระจก/สี/มือจับ → ระบบคิดส่วนต่างราคาให้ (เช่น อัปเป็นเทมเปอร์ +X,XXX) ลูกค้าเทียบได้ในใบเดียว

⚠️ ข้อควรระวัง

  • บานเปิดยูโร: ช่อง “กว้าง” กรอกกว้างรวมทั้งช่อง + จำนวนบาน (ระบบหารให้เอง)
  • สีชุบ: ต้องติ๊ก “ลูกค้าเห็นตัวอย่างจริงแล้ว” ก่อน
  • ตัวเลขราคาทุกตัวคิดจากระบบ — แก้มือเฉพาะกรณีจำเป็น
