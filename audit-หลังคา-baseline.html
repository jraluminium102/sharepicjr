ตารางเทียบรายการ (หมวดหลังคา)

รายการ (ใบจริงเขียน) ใบจริงเขียนอะไร ระบบมี option + ออกครบไหม ขาด/ผิด + root cause
ชนิดวัสดุมุง "หลังคาไวนิล สีขาว" / "หลังคาดีไลท์ DG-2" / "หลังคาชินโคไลท์ รุ่น NT-332 สี Light Grey หนา 6 มม." ออกหัวข้อ + สีวัสดุมุง (ไวนิล/ดีไลท์/โพลีตัน) ✅ ไวนิล/ดีไลท์/โพลีตันมีสีให้เลือกครบ — ตรง
โครงสร้างหลังคา "โครงหลังคาอลูมิเนียม สีเทาซาฮาร่า" / "โครงอลูมิเนียม" ออก "โครงอลูมิเนียม{สี}" หรือ "โครงเหล็กชุบซิงค์" + แปคู่/เดี่ยว ✅ มี dropdown โครงสร้าง + แป ครบ ออกลงใบถูก
รางน้ำ (มี/ระบบ) "รางน้ำอลูมิเนียม ... + โซ่รางน้ำ ลายดอกทิวลิป สีขาว" ปลายหลังคา=รางน้ำ → ออก "รางน้ำ (ท่อ สี…)" หรือ "รางน้ำ (โซ่ลาย… สี…)" ✅ ระบบรางน้ำ ท่อ/โซ่ + สี/ลาย ออกลงใบครบ
ชินโคไลท์ — รุ่น/สี/ความหนา "หลังคาชินโคไลท์ รุ่น NT-332 สี Light Grey หนา 6 มม." ออกแค่ชื่อ product เช่น "หลังคาชินโคไลท์ 6 มม. รุ่น Heat Cut" — ไม่มีช่องเลือกสี/รหัสรุ่น NT 🟡 ชินโคไลท์ (imp15–20) ไม่มี o-roofcolor (มีเฉพาะไวนิล/ดีไลท์/โพลีตัน) — สี Light Grey/รหัส NT-332 พิมพ์เองไม่ได้
เมทัลชีท — สี/ความหนาฉนวน (ใบจริงเมทัลชีทมักระบุสี/ความหนาฉนวน) (เช็คซ้ำ — ตัวอย่างใบเมทัลชีทมีน้อยในชุดนี้) ออกชื่อ product (ท้องPVC/เหล็ก EPS/PU) แต่ไม่มีช่องเลือกสีแผ่น 🟡 imp7–14 ไม่มี o-roofcolor — สีแผ่นเมทัลชีทระบุลงใบไม่ได้ (เช็คซ้ำกับใบจริงเมทัลชีท)
หลังคากระจกลามิเนต — สเปกกระจก/ฟิล์ม "หลังคากระจกลามิเนตใส ฟิล์มกันร้อน หนา 4+4 มม." / "ฟิล์มใส/เขียว 4+4" ออกแค่ "หลังคากระจกลามิเนต" — ไม่มีความหนา/ชนิดฟิล์ม 🔴 product roof_laminate ไม่มี glass selector / ไม่มี option ฟิล์ม-ความหนา · genQuote (บรรทัด 3224) ตัดคำ "หลังคา" ออกแล้วต่อชื่อ ทำให้เหลือสเปกเปล่า
ตะแกรงกันใบไม้ (รางน้ำ) "+ ตะแกรงกันใบไม้" / "ตะแกรงพลาสติกกันใบไม้ สีดำ" (เกือบทุกใบที่มีรางน้ำ) ไม่มี option · ไม่ออกลงใบ 🔴 ไม่มี product/option "ตะแกรงกันใบไม้" เลย (grep ทั้งไฟล์ไม่เจอ) — root: ขาด option
ท่อน้ำทิ้ง PVC (downpipe) "มีท่อน้ำทิ้ง PVC ปล่อยปลาย" / "ท่อน้ำทิ้ง PVC หนา 2.5 นิ้ว สีดำ/ขาว" มีแต่ "ครอบท่อ PVC" (o-rfpvc) ซึ่งเป็น ปลอกครอบ คนละอย่าง · ไม่ออกลงใบ 🔴 ขาด option ท่อน้ำทิ้ง PVC + ความหนา/สี · ของที่มี (ครอบ PVC) ก็ไม่ดันลงรายละเอียดงาน (calc-only) — root: ขาด option + ไม่ลงใบ
เสริมเพลทเหล็กล่าง / เหล็กดึงบน "เสริมเพลทเหล็กด้านล่างและเหล็กดึงด้านบน" มี checkbox o-rfplate/o-rfpull และ genQuote เขียนลงใบ (บรรทัด 3228–3229) 🟡 มี option + ลงใบได้ แต่ harness ไม่ติ๊ก checkbox จึงไม่เห็นใน render · ของจริงทำงาน (เช็คซ้ำในแอป) — จัดเป็น 🟡 เพราะยืนยันได้แค่จากโค้ด
เสาแผง (เสริมเหล็กภายใน) จำนวน X ต้น "เสาแผง ( เสริมเหล็กภายใน ) จำนวน 2 ต้น" (โชว์ใน detail) มีช่องกรอกจำนวน o-rfpolep + คิดราคา 4,000/ต้น แต่ ไม่เขียนลงรายละเอียดงาน 🔴 genQuote roof builder (3197–3231) ไม่ append เสาแผง — โชว์เฉพาะใน msgs (debug) · root: กรอกแล้วไม่ดึงลงใบ
คานเหล็กถัก "แขนค้ำอลูมิเนียม" / คานรับโครง (พบในงานยื่น) มี o-rfbeam + ราคา แต่ไม่เขียนลงใบ 🔴 เหมือนเสาแผง — calc-only ไม่ลงรายละเอียดงาน
โซ่รางน้ำ (ของเสริมแยก) "โซ่รางน้ำ ลายดอกทิวลิป สีขาว จำนวน 1 จุด" ระบบมี 2 ทาง: (ก) ระบบรางน้ำ=โซ่ → ออกลงใบ ✅ (ข) o-rfchain นับเส้น → ไม่ลงใบ 🟡 ทางหลัก (รางน้ำ=โซ่) ลงใบได้ แต่ option "จำนวนจุด/เส้น" (rfchain) ไม่ลงใบ — ซ้ำซ้อน/สับสน
ซ่อนสโลป (คอมโพสิต/สมาร์ทบอร์ด) (งานปิดสโลปใต้หลังคา — พบในงานกันสาดต่อชาย) มี o-rfhs + คิดราคา แต่ไม่เขียนลงใบ 🔴 calc-only ไม่ลงรายละเอียดงาน
ปิดปลายกันน้ำ / ปิดซ่อนราง / ครอบคาน "โครงปิดปลาย กันน้ำกระเด็น" / งานครอบปิดราง มีช่องกรอก (rfeave/rfgc/rfbc) + ราคา แต่ไม่เขียนลงใบ 🔴 calc-only ไม่ลงรายละเอียดงาน
ยื่นปลาย / ปล่อยปลาย / ไม่มีรางน้ำ "หลังคากระจกนิรภัยลามิเนต ยื่นปลาย ... ปล่อยปลาย/ไม่มีรางน้ำ" มีแค่ ปลายหลังคา = ปล่อย/รางน้ำ — ไม่มีคำ "ยื่นปลาย" และไม่เขียน "ไม่มีรางน้ำ" เมื่อปล่อย 🔴 genQuote (3210) ถ้า rfend=ปล่อย → ไม่เขียนบรรทัดอะไรเลย · ใบจริงระบุชัดว่า "ปล่อยปลาย/ไม่มีรางน้ำ" — root: ออกไม่ครบคำ
Skylight (ช่องแสงบนหลังคา) หัวข้อ "Skylight ห้องนอนใหญ่ ชั้น 2" + หลังคากระจกลามิเนต ไม่มี product Skylight แยก (ใช้ roof_laminate ตั้งชื่อหัวข้อเอง) 🔴 ไม่มี product/preset "Skylight" — ต้องเลือกหลังคาลามิเนตแล้วเปลี่ยนชื่อหัวเอง · ใช้ได้แต่ไม่มี template (เช็คซ้ำว่าจำเป็นแยกไหม)

หลักฐาน (จากใบจริง vs ระบบ render)

ใบจริง — คุณธนบัตร (QT2023100045) รายการ 1 หลังคาหน้าบ้าน

- หลังคาไวนิล โครงอลูมิเนียม เสริมเพลทเหล็กด้านล่างและเหล็กดึงด้านบน พร้อมรางน้ำอลูมิเนียม + ตะแกรงกันใบไม้ และมีท่อน้ำทิ้ง PVC ปล่อยปลาย
- เสาแผง ( เสริมเหล็กภายใน ) จำนวน 2 ต้น
รายละเอียดงาน: - หลังคาไวนิล สีขาว · - รางน้ำอลูมิเนียม + ตะแกรงพลาสติกกันใบไม้ สีดำ · - ท่อน้ำทิ้ง PVC หนา 2.5 นิ้ว สีดำ · - เสาแผง จำนวน 2 ต้น

ระบบ render — quote-FULL-B รายการ 43 (หลังคา ไวนิล)

หลังคา ไวนิล (3.6 × 2.4 ม.) · รายละเอียดงาน: - หลังคาไวนิล สีเทา · - โครงเหล็กชุบซิงค์ (แปคู่) · - รางน้ำ (โซ่ลายโลตัส สีน้ำตาล)
→ ขาด: ตะแกรงกันใบไม้ · ท่อน้ำทิ้ง PVC · เสาแผง · เสริมเพลท/เหล็กดึง

ใบจริง — คุณชวลิต (QT2025070050) รายการ 3/8 หลังคากระจกลามิเนต

- หลังคากระจกนิรภัยลามิเนต ยื่นปลาย โครงอลูมิเนียม ปล่อยปลาย/ไม่มีรางน้ำ
รายละเอียดงาน: - หลังคากระจกลามิเนตใส ฟิล์มกันร้อน หนา 4+4 มม.

หลังคา กระจกลามิเนต (1.2 × 1.2 ม.) · รายละเอียดงาน: - หลังคากระจกลามิเนต (ไม่มีความหนา/ฟิล์ม) · - โครงเหล็กชุบซิงค์ (แปคู่) · - รางน้ำ (โซ่ลายโลตัส สีน้ำตาล)
→ ขาด: ความหนา 4+4 มม. · ชนิดฟิล์ม (กันร้อน/ใส/เขียว) · คำ "ยื่นปลาย / ปล่อยปลาย-ไม่มีรางน้ำ"

สรุปสิ่งที่ต้องเพิ่ม/แก้ (จัดลำดับความสำคัญ)

  • P1ดันของเสริมหลังคาลงรายละเอียดงาน — genQuote roof builder (บรรทัด ~3197–3231) ปัจจุบันเขียนเฉพาะ โครง/แป/รางน้ำ/เพลท/เหล็กดึง · ต้อง append: เสาแผง (จำนวนต้น), คานเหล็กถัก, ซ่อนสโลป, ปิดปลายกันน้ำ, โซ่รางน้ำ(จุด), ปิดซ่อนราง, ครอบคาน — ของพวกนี้คิดเงินแล้ว (บรรทัด 1401) แต่ไหลแค่ลง msgs ไม่ลงใบ (กระทบทุกใบที่มีของเสริม)
  • P1เพิ่ม "ตะแกรงกันใบไม้" + "ท่อน้ำทิ้ง PVC (ความหนา/สี)" — ใบจริงเขียนแทบทุกงานที่มีรางน้ำ แต่ระบบไม่มี option เลย (มีแค่ "ครอบท่อ PVC" คนละอย่าง)
  • P1หลังคากระจกลามิเนต: เพิ่มสเปกกระจก/ฟิล์ม — option ความหนา (4+4/5+5/6+6 มม.) + ชนิดฟิล์ม (ใส/เขียว/กันร้อน) ให้ออกเป็น "หลังคากระจกลามิเนตใส ฟิล์มกันร้อน หนา 4+4 มม." ตามใบจริง
  • P2เพิ่มช่องเลือกสี/รหัสรุ่นให้ ชินโคไลท์ (imp15–20) และ เมทัลชีท (imp7–14) — ใบจริงระบุ "รุ่น NT-332 สี Light Grey" · ปัจจุบันมี o-roofcolor เฉพาะไวนิล/ดีไลท์/โพลีตัน (เมทัลชีท: เช็คซ้ำกับใบจริงเมทัลชีทว่าระบุสีอะไรบ้าง)
  • P2เติมคำ "ยื่นปลาย" และเขียน "ปล่อยปลาย / ไม่มีรางน้ำ" ชัดเจน — ตอนนี้ rfend=ปล่อย ระบบไม่เขียนบรรทัดเลย (3210) ทำให้ใบดูข้อมูลขาดเทียบใบจริง
  • P3พิจารณา preset "Skylight" — ปัจจุบันใช้ roof_laminate แล้วเปลี่ยนชื่อหัวเอง ใช้ได้ แต่ไม่มี template ช่วย (เช็คซ้ำว่าจำเป็นแยก product ไหม)
  • P3ลดความซ้ำซ้อนโซ่รางน้ำ — มี 2 ทาง (ระบบรางน้ำ=โซ่ ลงใบได้ / o-rfchain นับเส้น ไม่ลงใบ) ควรรวมหรือทำให้ rfchain ลงใบด้วย

1. harness enableAllOpts ติ๊กเฉพาะ dropdown ไม่ติ๊ก checkbox/ไม่กรอกตัวเลข → เสริมเพลท/เหล็กดึง/เสาแผง จึงไม่โผล่ใน render · แต่ข้อสรุป "ของเสริมไม่ลงใบ" ยืนยันจากโค้ด genQuote โดยตรง (builder ไม่ append) — ไม่ได้สรุปจาก render อย่างเดียว
