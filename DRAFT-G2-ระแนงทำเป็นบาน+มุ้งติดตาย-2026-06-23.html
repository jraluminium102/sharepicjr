หลักการ "ทำเป็นบาน"
• ระแนงสลับเป็น ไม่โครง อัตโนมัติ (บานให้โครงเอง · กันคิดซ้ำ +12k)
• ราคารวม = ค่าระแนงไม่โครง + ค่าทำบาน (doorStructBase · สูตรบานจริงเหมือน G1)
• ออปชั่นบาน (ล็อก/สีบาน) = โผล่เมื่อติ๊ก "ทำเป็นบาน" เท่านั้น
เคาะแล้ว (23 มิ.ย. รอบ 3):
• รางล่าง (SMS/ยูโร) = มีออปชั่นราง "รางกันน้ำ (มาตรฐาน) / รางเตี้ย 7มม." คืนมาแล้ว
• บานเลื่อนรางบน = ไม่มีออปชั่นราง (ไม่มีรางล่าง)
• ตัด กระจกบาน / คาดตาราง / สีอลูบาน ออก — ระแนงไม่มีกระจก
• ค่าทำสีบาน = เรตเท่าบานปกติ door 30% (ไม่ต่าง) · อยู่ใน details พับ
• มุ้งติดตาย ย้ายขึ้นบนก่อน "งานเสริม" · โชว์ตลอด (toggle อิสระ ไม่ผูกทำเป็นบาน)
• เพิ่ม ผ้ามุ้ง 3 แบบ: ไฟเบอร์ (มาตรฐาน) / กันแมว (+300 DEMO) / กันหนูสแตนเลส (+800 DEMO)
• ชนิดบาน 5 ปุ่ม + ปุ่มเปิด/ปิดคัดบาน คงไว้เหมือนเดิม
